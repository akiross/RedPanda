--------------------------------------

> sds('file-path', commentstring=None, delimiter=None, numlines=20, skipinitialspace=True)
> open(datasetname='reader', [scelta di cosa caricare])

> view(datasetname='reader', [scelta di come effettuare la visualizzazione])
> display()
